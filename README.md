### 작업 내용
1. onetime : socket을 사용한 1회성 채팅 앱
2. multi_use : socket을 사용해 다회성 채팅이 가능
    - ver1 : 편지로 주고 받듯이, 순서가 정해져있고, 서로 무조건 한번씩만 주고 받아야 한다. 서로 대화하는 것이 불가능하다.
    - ver2 : 우리가 생각하는 채팅형태. 원하는 만큼 메시지를 전송/수신할 수 있음.
